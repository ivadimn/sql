4. Для лайков всех объектов можно предложить следующую структру:

CREATE TABLE types_objects (
  id INT UNSIGNED NOT NULL AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(128) NOT NULL ,
  table_name VARCHAR(128) NOT NULL
);

CREATE TABLE likes_objects (
  id INT UNSIGNED NOT NULL AUTO_INCREMENT PRIMARY KEY,
  type_id INT UNSIGNED NOT NULL,
  entity_id INT UNSIGNED NOT NULL
);

CREATE TABLE likes (
  id INT UNSIGNED NOT NULL AUTO_INCREMENT PRIMARY KEY,
  user_id INT UNSIGNED NOT NULL ,
  object_id INT UNSIGNED NOT NULL ,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
Талица types_objects содержит типы объектов которые можно лайкать.
Таблица likes_objects содержит все сущности entity_id, которые лайкнули с указанием его типа type_id
(недостаток в том что en)
